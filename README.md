# BTK Akademi HTML5 Web Geliştirme - Demo Site

Bu repo BTK Akademi'nin **HTML5 ile Web Geliştirme** eğitiminde öğrendiklerimle oluşturduğum kişisel tanıtım web sitesini barındırıyor. Proje, temel HTML yapısı, CSS ile stil verme, kart yerleşimleri ve iletişim formu düzenlerini bir arada kullanarak eğitim sırasında pekiştirdiğim kavramları canlıya taşıyor.

## Öne çıkan bölümler
- **Üst gezinme**: Sabit pozisyonda duran başlık ve sekmelerle tek sayfa içinde hızlı ulaşım.
- **Hakkımda / profil kartı**: Yuvarlak avatar alanı, kısa akademik ve hedef odaklı biyografi.
- **Projeler**: Tablo düzeninde iki kart; her biri proje açıklaması ve dış bağlantı butonu içeriyor.
- **İletişim**: Aynı sayfadan mesaj gönderme izlenimi veren inputlar ve gönder/temizle butonları.
- **Sosyal bağlantılar**: Footer'da Remix Icon ile e-posta, LinkedIn, GitHub, Spotify, X ve WhatsApp ikonları.

## Kullanılan teknolojiler
- Saf HTML5 (semantik bölümler, tablolar, butonlar)
- CSS (renk paleti #303481, #D6E6F2, #F5F5F5, #FFF200 tonları; sabit header; kutu gölgelendirme; form stilleri)
- Remix Icon kütüphanesi (ikonları kolayca kullanmak için CDN üzerinden eklenmiştir)
